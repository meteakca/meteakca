# meteakcaoglu.com

Personal academic website for Mete Akcaoglu — built with [Quarto](https://quarto.org) and deployed to GitHub Pages.

## Local development

```bash
quarto preview
```

## Deployment

Pushes to `master` trigger a GitHub Actions workflow that renders the site and deploys to GitHub Pages at [meteakcaoglu.com](https://meteakcaoglu.com).
